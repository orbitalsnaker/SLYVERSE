## Características Generales

### IA GRATUITA ETERNA
- **100 % offline:** Todo el motor de IA se encuentra dentro del archivo HTML.
- **Sin límites:** Uso ilimitado, sin tiempo ni cantidad de consultas.
- **Sin registro:** No se requiere iniciar sesión ni aportar datos personales.
- **Sin anuncios:** Interfaz limpia y enfocada en la conversación.

### Interacción sencilla
- Presiona **cualquier tecla** para abrir el chat.
- Pulsa **Enter** para enviar tu pregunta.
- Presiona **ESC** para cerrar el chat.
