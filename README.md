## Características Generales

### IA GRATUITA ETERNA  
- **100 % offline**: Todo el motor está contenido dentro del archivo HTML.  
- **Sin límites**: No hay restricciones de uso ni de tiempo.  
- **Sin registro**: No necesitas iniciar sesión ni dar datos.  
- **Sin anuncios**: Interfaz limpia para enfocarte solo en la conversación.

### Interacción sencilla  
- Presiona **cualquier tecla** → se abre el chat.  
- Pulsa **Enter** para enviar una pregunta.  
- Presiona **ESC** para cerrar el chat en cualquier momento.

### Áreas de consulta  
Puedes preguntarle sobre prácticamente **todo**, incluyendo:  
- Amor y relaciones  
- Física, matemáticas y filosofía del universo  
- Bitcoin, criptomonedas y tecnología blockchain  
- La Plantilla Silenciosa (su historia, significado y secretos)  
- Reflexiones sobre tu futuro  
- Recetas, cocina y vida cotidiana
